# Content Ideas

Search the web for current trending topics, news, and discussions in the user's specified niche or industry.

Then generate actionable content ideas:

1. **3 trending topics** — what people are talking about right now
2. **For each topic, provide:**
   - A specific content angle (not generic)
   - A catchy title/headline
   - Key points to cover
   - Why it's timely (what triggered the trend)
   - Target audience

3. **1 contrarian take** — a topic where you could offer a different perspective from the mainstream

4. **1 evergreen idea** — something not time-sensitive but relevant to the niche

For each idea, suggest the best format: blog post, video, tweet thread, newsletter, or short-form video.

Base recommendations on actual current data from web search, not generic suggestions.
